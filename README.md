# Mongod

Simple Object-relational mapping gem for Mongo DB on Rails 4.
Pagination enabled.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'mongod'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install mongod
